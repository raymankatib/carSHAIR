## To run the App on your local machin

### 1 - Navigate to the App dirctory

### 2 - Run : `npm install`

### 3 - Run : `npm start`

### 4 - App should be running on port 3000

## For live demo for the app plase visit [Live Demo](http://localhost:3000)
